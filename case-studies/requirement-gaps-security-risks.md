# How Requirement Gaps Become Security Risks

## Scenario

A business team requests a new feature.

The request is approved and implemented.

Several weeks later it becomes clear that access controls were not fully considered.

## Root Cause

The issue was not technical.

The issue was incomplete requirements gathering.

## Security Impact

- Excessive permissions
- Data exposure
- Audit findings
- Increased operational risk

## Recommended Controls

- Requirement validation
- Risk review checkpoints
- Access review process
- Security stakeholder involvement

## Functional Analyst Perspective

Many security issues originate long before implementation.

They begin during requirement definition and stakeholder alignment.
